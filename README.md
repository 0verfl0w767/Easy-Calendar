# Easy-Calendar

✅ 월 변경 (화살표 클릭으로 자유롭게 변경 가능)

✅ 일정 확인 (달력 표로 확인 가능)

✅ 일정 추가 (일정 제목, 기간, 색깔 설정 가능)

✅ 일정 삭제 (일정 클릭 시 삭제 가능)

✅ 관리자 비밀번호 (추가, 삭제시 관리자 비밀번호 확인)
